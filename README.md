# CareerPath（仮）

資格取得まで、あと何時間？  
**合格への逆算ロードマップ × 学習管理アプリ**

---

## 機能

| 画面 | 機能 |
|------|------|
| **ランディングページ** | アプリ紹介・コンセプト説明 |
| **ダッシュボード** | 学習目標進捗・タイマー・週間コミット確認 |
| **目標管理** | 資格の追加・削除・進捗確認（逆算ロードマップ） |
| **週間コミット** | 今週の目標時間を宣言・棒グラフで日別記録確認 |
| **カレンダー** | Githubライクな学習ヒートマップ・セッション履歴 |

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm start
```

ブラウザで http://localhost:3000 を開く。

## ビルド

```bash
npm run build
```

`build/` フォルダに本番用ファイルが生成されます。

## 技術スタック

- **React 18** + **TypeScript**
- **React Router** (画面遷移)
- **localStorage** (データ永続化 ※サーバー不要)
- CSS (カスタムプロパティ)

## プロジェクト構造

```
src/
├── types/          # TypeScript型定義
├── data/           # 資格・キャリアマスターデータ
├── hooks/          # useAppState（状態管理）
├── pages/
│   ├── Landing.tsx    # LPページ
│   ├── Dashboard.tsx  # ホーム
│   ├── Goals.tsx      # 目標管理
│   ├── Commit.tsx     # 週間コミット
│   └── Calendar.tsx   # カレンダー
└── App.tsx         # ルーティング
```
